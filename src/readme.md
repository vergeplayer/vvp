#源码
## 目录结构说明


### scripts
js
----
 #### base:  基础公共类
 #### utils: 工具方法
 #### view:  视图层 DOM组件相关
 #### plugins: 插件
 #### service: 业务相关


### stylesheets
sass
----
 #### variables: 基础变量
 #### utilities: 工具方法
 #### mixins: 函数方法
 #### components: 组件


### templates
jade
----
 #### components: 组件
 #### layout: 结构框架
 #### plugin: 插件
 #### service: 优土业务逻辑
